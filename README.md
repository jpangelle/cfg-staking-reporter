# CFG Staking Reporter

Reports CFG Gains Daily via Discord

### Contributing

#### Clone the repo

```sh
git clone git@github.com:jpangelle/cfg-staking-reporter.git
```

#### Install dependencies

```sh
yarn install
```

#### Add env variables

```sh
cp example.env .env
```

#### Run app in development mode

```sh
yarn dev
```

#### Run linter

```sh
yarn lint
```

#### Build app

```sh
yarn build
```

#### Run in production mode

```sh
yarn start
```
